## Installation

To get started with PDF-AI, follow these steps:

1. Clone the repository:
    ```bash
    git clone https://github.com/KademEray/PDF-AI.git
    ```
2. Navigate to the project directory:
    ```bash
    cd PDF-AI
    ```
3. start the application:
    ```bash
    docker-compose up
    ```
